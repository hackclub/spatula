spatula!

a ysws program from hack club

Spatula is a Hack Club YSWS program (only for high schoolers and younger, ≤18) wherein you ship an app or script that sends push notifications (via Ntfy.sh) after fetching data and, assuming you work for ~1 hour, we'll give you a $5 grant card for Twilio (good for 1000+ SMS messages) so you can start sending SMS messages with your code!

Spatula ends on June 27, 2025 at 5:00 PM ET

How it works

  1. Build and ship a script that sends push notifications using Ntfy.sh.
  2. Spend about an hour working on your project.
  3. Submit your project to Hack Club YSWS.
  4. Receive a $5 HCB card for Twilio (enough for 1000+ SMS messages) to start sending SMS with your code!
